### spring-cloud-alibaba微服务实战案例分享
基于spring-cloud-alibaba套件的微服务架构的商场停车场实战案例

关键技术点应用：

服务注册. 服务发现——nacos
分布式配置——nacos
分布式锁——redission
分布式事务——seata
接口限流. 熔断. 降级——sentinel
统一网关——spring-cloud-gateway
代码简化——lombok,mybatis-generator,mybatis-plus
异步消息处理——rocketmq
分布式缓存——redis
客户端负载均衡——openfeign
RPC调用——dubbo
